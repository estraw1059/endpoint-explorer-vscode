# Call an endpoint

Click any endpoint in the **Endpoints** list. A Postman-style editor opens, pre-filled with everything Claude found:

- **Params** — query parameters (checkbox = include in the URL)
- **Headers** — request headers; your Authorization panel headers are added automatically on send
- **Path Variables** — values that replace `{id}` / `:id` placeholders in the URL
- **Body** — an example JSON body for POST/PUT/PATCH routes

Tweak anything, then hit **Send**. Status, timing, response headers and a pretty-printed body appear below.
